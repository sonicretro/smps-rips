General Pointer List:	8000 (SegaNet) (Z80 Bank Offset) / 1300 (Eraser) (Z80 RAM) (J)
Music Pointer List:	822A (SegaNet) (Z80 Bank Offset) / 1347 (Eraser) (Z80 RAM) (J)
SFX Pointer List:	8232 (SegaNet) (Z80 Bank Offset) / 1000 (Eraser) (Z80 RAM) (J)
Mod. Pointer List:	800E (SegaNet) (Z80 Bank Offset) / -- invalid -- (Eraser, points to 130E) (Z80 RAM) (J)
PSG Pointer List:	8154 (SegaNet) (Z80 Bank Offset) / 1310 (Eraser) (Z80 RAM) (J)
Z80 Snd Driver Offset:	00956E (SegaNet) / 020132 (Eraser) (J)
Additional Z80 Data #1:	021118 (Eraser) (J) (goes to Z80 RAM 1000)
Additional Z80 Data #2:	0213D3 (Eraser) (J) (goes to Z80 RAM 1300)
Pan Animation List:	02F6 (Z80 RAM)
FM Drum Pointer List:	0A7A (Z80 RAM)
FM Drum Ins. Ptr. List:	0A8E (Z80 RAM)
PSG Drum Pointer List:	0A60 (Z80 RAM)
Sound RAM:		1C00 (Z80 RAM)


There is only 1 music bank: 008000


Notes:
- The ROM contains 2 SMPS drivers. The first one (SegaNet) is loaded by the code at 0035AA and will be overwritten by the second one a short time later.
  The second driver (Eraser) is loaded by code at RAM offset 1CCE, which is copied there from ROM offset 021CCE.
  The information for the second loading routine is stored at ROM offset 020120.
	Size	Z80Ofs	ROMOfs
	0FE5	0000	0132
	02BA	1000	1118
	08D8	1300	13D3
- The FM drums of the two drivers differ slightly, but the sound code matches exactly.
- For the SegaNet music and dumped driver, see the "Sega Game Toshokan" folder.
- The drums use Modulation data from the SegaNet offset list. (Sonic Eraser lacks Modulation data)
